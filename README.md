# BesmokeInventoryApp

## Prerequisites
- Node.js
- npm
- .NET SDK

## Running the Client
```bash
cd besmokeinventoryapp.client
npm install
npm run dev
```

## Running the Server
```bash
dotnet restore
dotnet run --project BesmokeInventoryApp.Server
```

## Running Checks
```bash
npm run lint
dotnet test
```
